## Project Structure

`src` is the actual project directory. Thus open `src` as another project in IDE too. Then 
set `src`'s `PROJECT GOPATH` to `GDTS/gopath`, and edit `GDTS/gopath/src/config/config.go`

## Environment

- Go SDK 1.10.8
- etcd (cluster) v3.3.8
